C√≥mo a√±adir elementos a un gr√°fico

  • Usando points y abline

  • Otro ejemplo
    • Aplicamos la funci√≥n \(x^2+2x+\sqrt{\lvert x \rvert}\)
f <- function(x){
  x^2 - 2*x + sqrt(abs(x))
}

plot(f, xlim = c(-3,3))
points(0,0, pch = 19)  # a√±adimos un punto
points(-3:3, (-3:3)^2, col = "blue") # a√±adimos varios puntos
abline(2,3, lty = "dashed", col = "red") # a√±adimos una recta
abline(v = 2, lty = "dotted", col ="green")
abline(h = 5, lty = "dotdash", col = "gray")
